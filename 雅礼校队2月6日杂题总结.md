```cpp
#include <bits/stdc++.h>  
using namespace std;
int main() {
    int t;
    cin >> t; 

    while (t--) {  
        int n;
        cin >> n;  

        int a[1001];

        for (int i = 0; i < n; i++) {
            cin >> a[i];  
        }

        int max_num = a[0]; 
        for (int i = 1; i < n; i++) {
            if (a[i] > max_num) {
                max_num = a[i];  
            }
        }

        cout << max_num * n << "\n";
    }

    return 0;
}
```
**思路大概一样，但这个方法，思路更清晰。**
思路：
1. 用普通数组存储数字；
2. 普通for循环读取数字；
3. 手动遍历找最大值。
